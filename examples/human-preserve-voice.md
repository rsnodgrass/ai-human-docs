# Getting Started

Welcome! This guide gets you running in under 5 minutes.

## Install

```bash
npm install mypackage
```

## Configure

Create `config.json`:

```json
{
  "api_key": "your-key-here"
}
```

## Run

```bash
npx mypackage start
```

You should see "Server running on port 3000".
